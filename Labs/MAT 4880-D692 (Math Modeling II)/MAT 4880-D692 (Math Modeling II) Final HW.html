Chaos Simulation¶

Assumptions

$x_1; x_2 \text{ Are the State Variables}$

Given Continuous Dynamical System

$\cfrac{dx_1}{dt} = (a-1)x_1 + ax_1^2$

$\cfrac{dx_2}{dt} = x_1 - x_2$

Give Discrete Dynamical System

$\Delta x_1 = (a-1)x_1 + ax_1^2$

$\Delta x_2 = x_1 - x_2$

Objective

  1. Find equilibrium Points
    • Prove that equilibrium pts for the continuous dynamical system are stable at $x_1 = x_2 = (a-1) = a$ for any $a > 1$
    • Prove that equilibrium pts for the discrete dynamical system are stable at $x_1 = x_2 = \frac{(a-1)}{a}$ for any $a > 1$
  1. Run a simulation w/ various the values: $a = [1.5, 2.0, 2.5, 3.0, 3.5, 4.0]$
    • You can choose arbitrary stating points for each run

Simulations¶

def f1(x1, x2, a):
    return (a-1)*x1 - a*x1**2

def f2(x1, x2, a):
    return x1 - x2

def continuous(a, N, L):
    for i in range (N):
        x1_Old=L[-1][0]
        x2_Old=L[-1][1]
        x1_New=x1_Old + f1(x1_Old, x2_Old, a)
        x2_New=x2_Old + f2(x1_Old, x2_Old, a)
        L.append([x1_New,x2_New])
    return np.array(L)

def discrete(a, N, L, delT= 1):
    for i in range (N):
        x1_Old=L[-1][0]
        x2_Old=L[-1][1]
        x1_New=x1_Old + delT* f1(x1_Old, x2_Old, a)
        x2_New=x2_Old + delT* f2(x1_Old, x2_Old, a)
        L.append([x1_New,x2_New])
    return np.array(L)
